Strings in Python

1. Introduction to Strings

A string is a sequence of characters enclosed in single quotes (' ') or double quotes (" "). Strings are used to store text data. 

name = "Python"
print(name)

2. Creating Strings

str1 = "Hello"
str2 = 'World'

print(str1)
print(str2)

3. String Indexing

Each character in a string has a position called an index. Indexing starts from 0. 

text = "Python"

print(text[0])
print(text[2])

Output: P t

4. String Slicing

Slicing is used to extract a part of a string. 

text = "Python"

print(text[0:3])
print(text[2:5])

Output: Pyt tho

5. String Length

text = "Python"

print(len(text))

Output: 6

6. String Concatenation

Concatenation means joining two strings. 

first = "Hello"
second = "World"

print(first + " " + second)

7. Repetition Operator

text = "Hi "

print(text * 3)

Output: Hi Hi Hi

8. Common String Functions

Function Purpose
upper() Convert to Uppercase
lower() Convert to Lowercase
title() Title Case
replace() Replace Text
find() Find Position
count() Count Characters

9. upper() Function

text = "python"

print(text.upper())

Output: PYTHON

10. lower() Function

text = "PYTHON"

print(text.lower())

11. replace() Function

text = "I like Java"

print(text.replace("Java","Python"))

12. find() Function

text = "Python Programming"

print(text.find("Programming"))

13. count() Function

text = "banana"

print(text.count("a"))

Output: 3

14. Checking Membership

text = "Python"

print("P" in text)
print("x" in text)

15. Example Program: Reverse String

text = "Python"

print(text[::-1])

Output: nohtyP

16. Example Program: Count Vowels

text = "Education"

count = 0

for ch in text.lower():
    if ch in "aeiou":
        count += 1

print("Vowels =", count)

17. Example Program: Palindrome Check

text = input("Enter Word: ")

if text == text[::-1]:
    print("Palindrome")
else:
    print("Not Palindrome")

18. Applications of Strings

  • Text Processing
  • Search Engines
  • Chat Applications
  • Data Analysis
  • Web Development

19. Summary

Strings are one of the most important data types in Python. They are used to store and manipulate text using indexing, slicing and various built-in functions.
