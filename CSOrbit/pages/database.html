Database Management System (DBMS)

1. What is Data?

Data is a collection of facts, numbers, text, images or information that can be stored and processed by a computer.

Examples: Student names, marks, phone numbers and addresses.

2. What is a Database?

A database is an organized collection of related data stored electronically for easy access, management and updating.

3. Need for Databases

  • Store large amounts of data
  • Easy searching and retrieval
  • Reduce duplication of data
  • Improve security
  • Manage information efficiently

4. What is DBMS?

DBMS (Database Management System) is software used to create, manage, update and retrieve data from databases.

Examples: MySQL, Oracle, Microsoft SQL Server, PostgreSQL.

5. Advantages of DBMS

  • Data Security
  • Data Consistency
  • Data Sharing
  • Backup and Recovery
  • Reduced Data Redundancy

6. Components of a Database

  • Tables
  • Rows (Records)
  • Columns (Fields)
  • Primary Key
  • Relationships

7. Table Example

Student ID Name Class
101 Shivam 8
102 Rahul 8
103 Priya 8

8. What is SQL?

SQL (Structured Query Language) is used to communicate with databases. It helps users create, read, update and delete data.

9. SELECT Statement

SELECT * FROM Students;

Displays all records from the Students table.

10. INSERT Statement

INSERT INTO Students
VALUES(104,'Amit',8);

Adds a new record into the table.

11. UPDATE Statement

UPDATE Students
SET Name='Rohan'
WHERE StudentID=104;

Updates existing data in a table.

12. DELETE Statement

DELETE FROM Students
WHERE StudentID=104;

Deletes a record from the database.

13. Applications of Databases

  • Schools and Colleges
  • Banks
  • Hospitals
  • Railway Reservation Systems
  • E-Commerce Websites
  • Social Media Platforms

14. Database Security

  • Use strong passwords
  • Grant proper user permissions
  • Regular backups
  • Data encryption
  • Firewall protection

15. Summary

A Database Management System (DBMS) helps organize, store, manage and retrieve data efficiently. SQL is used to interact with databases and perform operations such as inserting, updating, deleting and retrieving records.
