HTML Basics

What is HTML?

HTML (HyperText Markup Language) is used to create web pages. It provides the structure of a website.

Basic HTML Structure

<!DOCTYPE html>
<html>
<head>
    <title>My Page</title>
</head>
<body>
    Content Here
</body>
</html>

Headings

<h1>Main Heading</h1>
<h2>Sub Heading</h2>

Paragraph

<p>This is a paragraph.</p>

Link

<a href="https://google.com">
Google
</a>

Image

<img src="image.jpg">

List

<ul>
  <li>HTML</li>
  <li>CSS</li>
</ul>

Summary

HTML is the standard language used to create web pages. It uses tags to structure content such as headings, paragraphs, images, links and lists.
