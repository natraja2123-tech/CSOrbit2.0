Fundamentals of Python

1. Introduction to Python

Python is a high-level, interpreted and easy-to-learn programming language. It was developed by Guido van Rossum and released in 1991.

2. Features of Python

  • Easy to Learn
  • Simple Syntax
  • Object-Oriented
  • Cross Platform
  • Open Source
  • Large Library Support

3. First Python Program

print("Hello World")

Output: Hello World

4. Variables

Variables are used to store data values. 

name = "Shivam"
age = 16

print(name)
print(age)

5. Data Types

  • int → Integer
  • float → Decimal Number
  • str → String
  • bool → True/False
  • list → Collection of Values
x = 10
y = 3.14
name = "Python"

6. Input from User

name = input("Enter Name: ")

print("Hello", name)

7. Operators

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulus
a = 10
b = 5

print(a+b)
print(a-b)
print(a*b)

8. Conditional Statements

Used to make decisions in programs. 

age = 18

if age >= 18:
    print("Adult")
else:
    print("Minor")

9. For Loop

for i in range(1,6):
    print(i)

Output: 1 2 3 4 5

10. While Loop

i = 1

while i <= 5:
    print(i)
    i += 1

11. Functions

Functions help reuse code. 

def greet():
    print("Welcome")

greet()

12. Lists

fruits = ["Apple","Banana","Mango"]

print(fruits[0])

13. Strings

text = "Python"

print(text.upper())
print(text.lower())

14. Example Program: Addition

a = int(input("Enter First Number: "))
b = int(input("Enter Second Number: "))

sum = a + b

print("Sum =", sum)

15. Example Program: Even or Odd

num = int(input("Enter Number: "))

if num % 2 == 0:
    print("Even")
else:
    print("Odd")

16. Example Program: Multiplication Table

num = 5

for i in range(1,11):
    print(num * i)

17. Applications of Python

  • Web Development
  • Artificial Intelligence
  • Machine Learning
  • Data Science
  • Game Development
  • Automation

18. Summary

Python is one of the most popular programming languages because of its simple syntax and powerful features. It is used in web development, AI, data science, automation and many other fields.
