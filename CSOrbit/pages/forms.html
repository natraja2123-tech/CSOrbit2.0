HTML Forms

What is a Form?

HTML Forms are used to collect information from users, such as names, emails and passwords.

Basic Form Example

<form>
  <input type="text">
  <input type="submit">
</form>

Input Types

  • Text
  • Email
  • Password
  • Radio Button
  • Checkbox
  • Submit Button

Summary

Forms are used to collect user information. The most common form elements are text boxes, email fields, password fields and submit buttons.
