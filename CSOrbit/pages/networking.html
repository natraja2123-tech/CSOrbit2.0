Networking Fundamentals

Introduction to Computer Networks

A computer network is a collection of computers and devices connected together for sharing information and resources.

Network Devices

  • Router
  • Switch
  • Hub
  • Modem

Types of Networks

  • LAN - Local Area Network
  • MAN - Metropolitan Area Network
  • WAN - Wide Area Network

Network Protocols

Protocols are rules used for communication. Examples: HTTP, HTTPS, FTP, TCP/IP.

Network Security

Use strong passwords, antivirus software, firewall and secure connections.
