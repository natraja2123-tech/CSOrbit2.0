Lists in Python

What is a List?

A List is a collection of items stored in square brackets [ ]. Lists can store numbers, strings and other values.

Creating a List

fruits = ["Apple","Banana","Mango"]

print(fruits)

Accessing List Items

fruits = ["Apple","Banana","Mango"]

print(fruits[0])

Output: Apple

Adding Items

fruits.append("Orange")

print(fruits)

Removing Items

fruits.remove("Banana")

print(fruits)

Length of List

print(len(fruits))

Example Program

numbers = [10,20,30,40]

for num in numbers:
    print(num)

Summary

Lists are used to store multiple values in one variable. They are mutable, which means their contents can be changed.
